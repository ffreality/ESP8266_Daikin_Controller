# ESP8266_Daikin_Controller
ESP8266 based Daikin Climate IR Remote Controller and Dallas DS1820 Temprature Sensor.

## FEATURES
- Turn on and off AC.
- Change temperature
- Get room temperature with sensor (it refreshes at 60 seconds)
- It has a web panel and API.
